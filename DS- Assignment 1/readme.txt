Code Overview:
Read the house price file in a DataFrame.
Process the data.
Normalize the data.
Remove the outliers.
Find the important features using correlation method.
Define Training and Test Set.
Define 'root mean square error', 'r2 score', and 'cross validation error' to compare various models.
Train various models on the training set.
Select the model with lowest cross validation error.
Predict the prices of test set data.

Cross validation error: is used for comparison because it tells the expected error of a new test set.
